Verify on-chain

The artifact's metadata is stored 100% on Ethereum. Calling tokenURI() returns a data:application/json envelope with the rendered HTML, an SVG thumbnail, and trait data — fully self-contained, no server required.

Method 1 — Etherscan
Open the SettleArtifact contract on Etherscan → Contract → Read Contract → expand tokenURI → enter ID N → click Query. Output is a data:application/json;base64,... URI. Paste it into a browser tab to see the raw JSON metadata. To view the animation: decode the JSON, then paste the animation_url value into a new tab.

Method 2 — Copy URI directly
Paste this into any browser address bar to see the raw JSON metadata returned by the contract (name, description, image, animation_url). Copy the animation_url value and paste it into a new tab to render the animation.
Method 3 — animationURI() (direct render)
Paste this directly into any browser address bar to render the animation — no JSON decode needed. Same HTML payload as the animation_url field inside tokenURI(), but standalone. (animationURI() is a Modulo extension, not part of ERC-721; marketplaces use tokenURI().)
